State Mutations by Agent

Supervisor

Reads: user_query

Writes:

  • route_type
  • route_reason
  • matched_kpi
  • extracted_filters
  • agent_logs (+)

KPI Agent

Reads: user_query, matched_kpi, extracted_filters

Writes:

  • kpi_results
  • final_answer
  • generated_sql
  • visualization_config
  • agent_logs (+)

Analyst Agent

Reads: user_query

Writes:

  • analyst_result
  • final_answer
  • generated_sql
  • agent_logs (+)

Validator

Reads: final_answer, kpi_results, analyst_result

Writes:

  • validation_passed
  • validation_notes
  • agent_logs (+)

Visualization

Reads: kpi_results, visualization_config

Writes:

  • visualization_config (update)
  • agent_logs (+)

Clarify

Reads: user_query

Writes:

  • final_answer

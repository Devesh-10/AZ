Sustainability Insight Agent (SIA)

Multi-Agent AI System for Natural Language Sustainability Analytics

LangGraph + AWS Bedrock + Claude

🚀 System Overview

The Sustainability Insight Agent (SIA) is a multi-agent AI system built on LangGraph that enables natural language queries against sustainability data. It uses AWS Bedrock Claude models for intent classification and response generation, with deterministic SQL execution via DuckDB.

💡 Structured Output Pattern

"LLM classifies WHAT, Code renders HOW"

1

User Query

"What are the greenhouse gas emissions by scope type?"

2

Intent Classification (LLM)

{ "intent": "GHG_BY_SCOPE", "time_type": "current_year", "geography": null }
3

SQL Template Lookup (Deterministic)

SELECT 'Scope 1' as SCOPE_TYPE, SUM(SCOPE1_SITE_ENERGY_TCO2_QUANTITY + ...) as EMISSIONS_TCO2 FROM GREENHOUSE_GAS_EMISSIONS_MONTHLY_SUMMARY WHERE REPORTING_YEAR_NUMBER = 2025 UNION ALL ...

🛠 Key Design Patterns

🎯 Semantic Routing

Embeddings match queries to KPIs/tables with keyword boosting and LLM fallback for ambiguous queries.

📝 Structured Output

LLM classifies intent (WHAT), code renders SQL (HOW). Eliminates SQL hallucination.

✅ Deterministic Execution

Pre-defined SQL templates with DuckDB in-memory queries ensure consistent, reproducible results.

📊 Multi-Agent Telemetry

Each agent logs execution details for full audit trail and timeline visualization.

📈 Conditional Visualization

Charts only for KPI queries with data. Skip visualization for text-only analyst responses.

⚡ Serverless Architecture

Lambda + API Gateway for auto-scaling, pay-per-use compute with CloudFront edge delivery.

Main steps to mobilise and build Agentic AI

A structured plan to build 4 AI agents, co-design Human+AI workflows, and enable change across R&D IT.

March – May 2026
Phase 1: Test Intelligence Agent

Objective

Build foundational testing automation for 3DP and BIKG, validating the agentic approach with high-impact MVP.

Activities

  • Conduct discovery workshops with 3DP, BIKG teams to validate pain points
  • Build LLM-powered test generation, synthetic data, similar failure detection
  • Integrate with 3DP (regulatory tests) and BIKG (KG query tests)

Outputs

  1. Test Intelligence Agent deployed
  2. 25% reduction in manual test effort
  3. Reusable prompts & templates
June – August 2026
Phase 2: Knowledge Library + Ingestion

Objective

Scale to knowledge management and data ingestion, enabling self-service docs and automated data onboarding.

Activities

  • Build RAG pipeline for document search and Q&A across platform docs
  • Build schema inference and auto-mapping for data ingestion
  • Design human-in-the-loop review workflows

Outputs

  1. Knowledge Library with docs indexed
  2. Ingestion Companion with auto-schema
  3. 50% faster answers, 25-30% faster ingestion
September – November 2026
Phase 3: Auto-Lineage Agent

Objective

Complete the data lifecycle with lineage tracking, enabling FAIR compliance and regulatory audit readiness.

Activities

  • Define lineage requirements for regulatory (3DP) and biological (BIKG) data
  • Build graph-based lineage extraction and visualization
  • Implement FAIR evidence generation and audit export

Outputs

  1. Auto-Lineage Agent deployed
  2. FAIR compliance evidence for 3DP
  3. Training materials for scaling
